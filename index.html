Main tenses in Irish

Temps Forme de base Structure Traduction (Français)
Présent "tá" (ex: "tá mé" = je suis) Base verbale ("tá") + sujet Indique une action ou un état qui se déroule actuellement ou de façon habituelle.
Passé "bhí" (ex: "bhí mé" = j'étais) Base verbale ("bhí") + sujet Indique une action ou un état qui a eu lieu dans le passé.
Futur "beidh" (ex: "beidh mé" = je serai) Base verbale ("beidh") + sujet Indique une action ou un état qui aura lieu dans le futur.
Conditionnel "bheadh" (ex: "bheadh mé" = je serais) Base verbale ("bheadh") + sujet Exprime une action qui se produirait sous certaines conditions.
Impératif "tabhair" (ex: "tabhair dom" = donne-moi) Base verbale à l'impératif ("tabhair") + complément direct Exprime un ordre ou une demande.
Présent progressif "ag" + verbe (ex: "tá mé ag ithe" = je suis en train de manger) Base verbale "tá" + "ag" + verbe au gérondif Exprime une action en cours au moment où l'on parle.
Passé progressif "ag" + verbe au passé (ex: "bhí mé ag ithe" = j'étais en train de manger) Base verbale "bhí" + "ag" + verbe au gérondif Exprime une action qui était en cours dans le passé.
Futur progressif "beidh" + "ag" + verbe (ex: "beidh mé ag ithe" = je serai en train de manger) Base verbale "beidh" + "ag" + verbe au gérondif Exprime une action qui sera en cours dans le futur.

Main verbs in Irish

Verbe Présent Passé Futur Conditionnel Traduction (Français)
"tá" (être) "tá mé" (je suis) "bhí mé" (j'étais) "beidh mé" (je serai) "bheadh mé" (je serais) Indique un état ou une action en cours ou habituel.
"obair" (travailler) "oibrím" (je travaille) "obair mé" (je travaillais) "obróidh mé" (je travaillerai) "obróinn mé" (je travaillerais) Exprime une action de travail ou d'effort.
"ithe" (manger) "tá mé ag ithe" (je mange) "bhí mé ag ithe" (je mangeais) "beidh mé ag ithe" (je mangerai) "bheadh mé ag ithe" (je mangerais) Exprime l'action de consommer de la nourriture.
"féachaint" (voir) "féachaim" (je vois) "chonaic mé" (j'ai vu) "feicfidh mé" (je verrai) "feicfinn mé" (je verrais) Exprime l'action de percevoir avec les yeux.
"téigh" (aller) "téim" (je vais) "chuaigh mé" (je suis allé) "rachaidh mé" (j'irai) "rachainn mé" (j'irais) Exprime l'action de se déplacer d'un endroit à un autre.
"duirt" (dire) "deirim" (je dis) "duirt mé" (j'ai dit) "déarfaidh mé" (je dirai) "déarfainn mé" (je dirais) Exprime l'action de communiquer par parole.
"teastaigh" (avoir besoin) "teastaíonn sé" (il a besoin) "theastaigh sé" (il avait besoin) "teastóidh sé" (il aura besoin) "teastódh sé" (il aurait besoin) Exprime la nécessité ou le besoin de quelque chose.
"ceistigh" (demander) "ceistím" (je demande) "cheistigh mé" (j'ai demandé) "ceistfidh mé" (je demanderai) "ceistfinn mé" (je demanderais) Exprime l'action de poser une question.
"abair" (dire) "abair" (je dis) "abair mé" (j'ai dit) "abairfidh mé" (je dirai) "abairfinn mé" (je dirais) Exprime l'action de déclarer ou de prononcer quelque chose.
